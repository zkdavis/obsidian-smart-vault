# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-19

### Added
- **Handwritten Notes (Vision)**: Auto-process images in `Inbox/Handwritten` with OCR.
- **Smart Chat**: RAG-based Q&A with context awareness.
- **Smart Formatting**: Grammar check, Zettelkasten analysis, Flashcard generation.
- **Smart Organization**: AI folder recommendations.
- **Agentic Tools**: "Insert paragraph", "Pending Tasks", "Summarize Note".
- **Performance**: Binary caching (MessagePack) for 2.4x faster loads.
- **Settings**: Comprehensive configuration for models, timeouts, and thresholds.

### Improved
- **UI/UX**: Polished chat interface, compact suggestion toolbar, and native Obsidian look-and-feel.
- **Stability**: Robust error handling, LLM timeouts, and retry logic.

## [0.1.0] - 2026-01-19

### Added
- Initial release of Smart Vault Organizer
- embeddings and intelligent link suggestions using local LLMs
- WASM-based implementation for performance
- Basic settings tab

# Contributing to Smart Vault Organizer

Thank you for your interest in contributing to Smart Vault Organizer!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/obsidian-smart-vault.git`
3. Install dependencies: `npm install`
4. Build the project: `npm run build`

## Development

- Run `npm run dev` to start the development build.
- This plugin uses Rust/WASM for core logic. Ensure you have Rust and `wasm-pack` installed.

## Pull Requests

1. Create a new branch for your feature or fix.
2. Commit your changes with clear messages.
3. Push to your fork and submit a Pull Request.
